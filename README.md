# rtc-ssh
SSH via WebRTC (pions) tunnel
### Usage

```
$ go get -u github.com/mxseba/rtc-ssh
$ cd $GOPATH/bin
```
### First run
```
$ ./rtc-ssh newkey
uuid xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
Signal OK
```
You can connect to a SSH session, copy uuid key and paste on the static page: https://www.sqs.io
